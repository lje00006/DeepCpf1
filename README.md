DeepCpf1
----

DeepCpf1 is a R package for Cpf1 guide RNA on-target and off-target activity predictions and encoding.

The latest (devel) version of DeepCpf1 requires£º 

R 3.3.3
mxnet 0.9.4
seqinr 3.4-5
stringr 1.2.0
IRanges 2.8.2

Reference: Jiesi Luo, Wei Chen and Xiaobo Zhou. Prediction of activity and specificity of CRISPR-Cpf1 using convolutional deep learning neural networks.
